# Downtown - Modern News Magazine

A modern, responsive news magazine website built with Next.js 16, featuring a clean design with multi-column layout, featured articles, latest news, and category-based content organization.

## Tech Stack

- Next.js 16.1.4
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Installation

### Prerequisites

- Node.js 18.x or higher
- npm, pnpm, bun or yarn

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/shamim1845/downtown.git
   cd downtown
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open browser**
   
   Navigate to [http://localhost:3000](http://localhost:3000)

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
